Using Dictionary from other programs

You can use Odia-English to look up words in most Windows programs. To do this you must have used the default installation program to run from the system tray.

One-click look up (using the mouse)

  1. Hold down the Ctrl key, right click with the mouse on the word you want to look up

Look up a selected word (using the keyboard)

  1. Select the word or collocation you want to look up.
    In most programs you can double click with the mouse to do this.

In some programs you can replace the selected word by pressing the Replace button in the Dictionary window after you have chosen an alternative. You can also do this in MS Word and rich edit controls using the one-click look up method.
