Technical Information

Odia -English will work with most windows programs. But you may wish to integrate Dictionary more closely with your programs.

Programmers: Using Odia -English from your programs

You can either use the command line, send a message to the system tray icon (which will show Dictionary as though the use had clicked on it on the system tray), or you can call the  Data.DLL that is installed to the Windows or Windows System folder.

Command Line (requires Odia -English version 2.2 or higher)

Rundll32 Data.DLL,ShowRunDLL word_to_lookup
